Data Visualization & Storytelling – Superstore Dataset

Objective
The goal of this project is to create meaningful visualizations and tell a story using sales data.

Tools Used
Power BI

Visualizations Created
* KPI Cards (Total Sales, Profit, Quantity)
* Sales by Category (Bar Chart)
* Profit by Category (Bar Chart)
* Sales by Region (Pie Chart)
* Sales Over Time (Line Chart)
* Profit by Region (Bar Chart)
* Slicer for filtering data

Key Insights
* Technology category has the highest sales
* West region contributes the most revenue
* Sales show a steady trend over time

Files Included
* Superstore.csv (dataset)
* Dashboard Report (PDF)

Outcome
Created an interactive dashboard that provides clear insights and helps in understanding business performance.
